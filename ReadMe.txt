Web History Management with Qt GUI

GUI used: Qt
Link: https://www.qt.io/download

Project Overview

This project implements a simple web history management system with a graphical user interface (GUI) using the Qt framework. The application allows users to manage their search history, including functionalities such as adding pages, removing pages, displaying the entire history, and more.

Functionality


Basic Features

Add Page: Add a webpage to the search history.
Remove Page: Remove a specific webpage from the search history.
Display History: View the entire search history.
Clear History: Remove all entries from the search history.
Additional Functionalities
The project introduces three additional functionalities:

Most Recent 3 Websites:
Displays the most recent three websites in the search history.

Go to Website:
Navigates to a specific website in the search history.

Clear Search History:
Clears the entire search history, providing a confirmation prompt.
